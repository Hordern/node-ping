Node-ping

Node uptime monitor

View project onGitHub

Node-ping is a simple app that monitors the availability of your websites.

Note: Node-ping uses GMail to send down-time notification emails, you are required to create an application specific password on your Gmail account and update the config.json file.

How it works

  1. Download the repo: git clone https://github.com/qawemlilo/node-monitor.

  2. Create an application specific password from your Gmail account and update the config.json file.

  3. List all websites that you want to monitor in websites.json and run node app command to start monitoring your websites.

Dependencies

  • Node Mailer - for sending emails
  • Node Monitor - an uptime event emitter that was abstracted from Node-ping.
